## Workflow * Environment - Juno on Ubuntu Trusty - Qemu 2.3 / libvirt 1.3.1 - Dedicated network for admin * When do we need live migration ? - Need to empty a compute node - Hardware maintenance - Software issue - Kernel upgrade - microcode update
## Workflow * A bug is triggered by live migration - Try to reproduce it in an Openstack dev environment - Find root cause - Check if an upstream fix is already existing - Cherry pick on our Openstack env - If not fixed upstream - Need to be fixed quickly - workaround - openstack patch
